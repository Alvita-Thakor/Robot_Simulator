# Robot Simulator

A Python-based robot simulator developed to practice Object-Oriented Programming (OOP), software design, and Python programming.

## Features

* Interactive command-line interface
* Robot movement on a 2D grid
* Boundary checking
* Obstacle placement
* Robot rotation (Left / Right)
* Grid visualization
* Input validation
* Modular project structure

## Project Structure

```
Robot_Simulator/
│
├── main.py        # Program entry point
├── robot.py       # Robot class
├── grid.py        # Grid class and obstacle handling
├── utils.py       # Utility functions
├── README.md
└── .gitignore
```

## Commands

| Command | Description                                        |
| ------- | -------------------------------------------------- |
| MOVE    | Move the robot one step forward                    |
| LEFT    | Rotate the robot 90° left                          |
| RIGHT   | Rotate the robot 90° right                         |
| REPORT  | Display the robot's current position and direction |
| EXIT    | Exit the simulator                                 |

## How to Run

Clone the repository:

```bash
git clone https://github.com/Alvita-Thakor/Robot_Simulator.git
```

Navigate to the project directory:

```bash
cd Robot_Simulator
```

Run the simulator:

```bash
python main.py
```
